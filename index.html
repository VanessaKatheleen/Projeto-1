O que é HTML?


Uma das tecnologias que ajudou a solidificar as bases da internet como a conhecemos hoje, suas evoluções se tornaram cada vez mais importantes para estruturar diversas páginas na web.

É graças ao HTML que diferentes componentes visuais são estruturados dentro de um conteúdo e que informações sobre a página são comunicadas aos navegadores.

O HTML também é uma linguagem extremamente versátil que foi se desenvolvendo ao longo do tempo, estando hoje em sua quinta versão, o HTML5.


Conhecimentos básicos para começar estruturar um site em HTML


  • HTML

HTML: Tudo que estiver entre "HTML" dará estrutura a página

HEAD: Pré carrega antes de exibir para o usuário

BODY: o que de fato aparece para o usuário na página

"<>" Esse símbolo significa que está abrindo a TAG anterior

"< / >" Esse símbolo significa que está fechando a TAG anterior


  • ATRIBUTOS DE TAGS

"< I >" - Texto fica em ítalico;

"< strong >" - negrito;

"< input type=”test” >" - Mostra uma caixa de texto na página;

ID = ex: "< strong id=”titulo” >" - Serve para identificar em qual elemento está mexendo, muito utilizado em java;

STYLE = ex: "< strong style= “titulo” >" - É a forma para formatar o HTML, são comandos pré definidos, troca de cor, tamanhos de fonte etc, serve para por CSS direto no elemento;

CLASS = ex: "< strong class = “titulo-principal” >" - Forma de patronizar tudo que estiver titulo e principal;

"< h1 >< /h1 >" - Título de maior valor hierárquico

"< h2 >< /h2 >"

"< h3 >< /h3 >"

"< h4 >< /h4 >"

"< h5 >< /h5 >"

"< h6 >< /h6 > - "Título de menor valor hierárquico;

"< p >< /p >" - Principal tag de texto, compõe um parágrafo;

"< span >< /span >" - Apesar de ter uma funcionalidade e características parecidas com os parágrafos, costumam ser utilizadas apenas para pequenas informações, como legendas de um formulário, legendas de uma imagem, etc. Também pode ser utilizada para formar um containe;

"< pre >< /pre>" - Tag utilizada para representar texto pré-formatado. Muito utilizada para inserir códigos;

"< b >< /b >" - Transforma o conteúdo em negrito;

"< br/ >" - Essa tag serve para dar espaço entre as linhas e não necessita de fechamento;

"< hr/ >" - Essa tag adiciona uma linha horizontal e não necessita de fechamento.
